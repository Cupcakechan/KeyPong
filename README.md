# ⌨️ KEY PONG

**Game 1 of the 20 Games Challenge** — a mechanical-keyboard-themed reskin of the classic *Pong*.

Single-player Pong where your paddle is a vertical **Spacebar** key, the opponent is a simple AI, and the **ball is a keyboard key that morphs into a different random key sprite on every bounce.**

---

## 🎮 Controls
| Action      | Keys              |
|-------------|-------------------|
| Paddle Up   | `W` / `↑`         |
| Paddle Down | `S` / `↓`         |
| Confirm     | `Enter` / Mouse   |

## 🏆 Win Condition
First to **11 points** wins.

## 🛠️ Tech
- **Engine:** Unity 6.3 LTS
- **Render:** Built-in 2D
- **Input:** New Input System
- **PPU:** 28 · **Reference Resolution:** 1920×1080

## 📦 Build Targets
- WebGL (itch.io) — primary
- Windows standalone — stretch
